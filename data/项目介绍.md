# MyAgent 项目介绍

## 概述

MyAgent 是一个基于 LangChain 和 LangGraph 框架构建的个人知识库问答智能体。它能够加载用户的本地文档，并基于文档内容回答问题。

## 技术栈

- **LLM**: DeepSeek V3.1，通过阿里云 DashScope 平台调用
- **框架**: LangChain + LangGraph
- **向量数据库**: FAISS（本地向量存储）
- **Embedding 模型**: DashScope text-embedding-v3（1024 维）
- **语言**: Python 3.11

## 核心功能

1. **工具调用**: Agent 能够自主决定使用哪个工具来回答问题
   - 日期时间查询
   - 数学计算（基于 AST 的安全求值）
   - 网络搜索（DuckDuckGo）
   - 知识库检索（RAG）

2. **RAG（检索增强生成）**: 加载本地文档（PDF/Markdown/TXT），建立向量索引，根据用户问题检索相关内容后生成回答

3. **Token 使用统计**: 每次对话后显示累计的 prompt tokens、completion tokens 和总 tokens

## 开发计划

项目采用渐进式开发策略，共分 6 个阶段：
- Phase 1: 基础 Agent（已完成）
- Phase 2: 加入 RAG
- Phase 3: 加入 Memory（对话记忆持久化）
- Phase 4: 深入 Agent 图（手动构建 LangGraph）
- Phase 5: 深入 RAG 管线（自定义分块、检索优化）
- Phase 6: 高级特性（流式输出、Web UI 等）
